Backoffice

Este es manual de usuario del backoffice.

Registro

Para comenzar a usar el backoffice, primero se necesita tener una cuenta como administrador. Para ello, se puede ingresar en la opcion de "Registrarse", y le pedira ingresar los siguientes datos:

  • E-mail
  • Contraseña
  • Repetir contraseña
  • Nombre
  • Apellido

Una vez que tiene la cuenta, puede iniciar la sesion.

Inicio de Sesion

Para iniciar sesion, solo necesita contar con

  • E-mail
  • Contraseña

Luego ingresando la opcion iniciar sesion podra comenzar a utilizar las funcionalidades del backoffice.

Lista de usuarios

Una vez ingresado, la primera pantalla que aparece es el listado de usuarios utilizando la aplicacion. En la tabla figura la informacion basica,los cuales son:

  • Email
  • Nombre
  • Apellido
  • Chofer (Si es chofer o no)
  • Bloqueado (Si esta bloqueado o no)

Si se ingresa la opcion de visualizar (el icono del ojo), se puede ver mas datos tales como:

  • Ratings
  • Datos del vehiculo (en caso de ser chofer)

Cabe destacar que los ratings como pasajero son distintos a los de chofer.

Metricas

Si se ingresa en la solapa de metricas (en el borde superior de la pantalla), se puede ver tres areas principales:

Registros

Corresponde a la cantidad de registros nuevos de usuarios desde una fecha indicada y la actual, y se puede filtrar segun el tipo de registro (segun identidad federada o email y contraseña).

Logins

Al igual que los registros, corresponde a la cantidad de logins de usuarios desde una fecha indicada y la actual, y se puede filtrar segun el tipo de cuenta (segun identidad federada o email y contraseña).

Bloqueados

Aca se muestra la cantidad numerica total de usuarios bloqueados que hay en el sistema.

Reglas de Cotizacion

Como indica el nombre, aca se puede establecer las reglas que se utilizaran para calcular el precio de un viaje. Cada campo indica cuanto va a afectar cada parametro al precio final. Cabe destacar que el precio se cotiza en Ethereum.
